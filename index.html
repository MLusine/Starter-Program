Software Developer Starter Program

The objective of the program is to teach the fundamentals of Computer Science and programming language essentials in six months, after which the students will choose a specialization based on their preferences. The next six months of the program are allocated to the specialization course (Android App Development, iOS App Development, Backend Web Development, etc.) and project development.

Program Description

The ideal graduate of the program is a strong intern or a junior developer specialized in one of the mentioned specialties with solid knowledge in Computer Systems, Algorithms and Data Structures, C++, Java, Object-Oriented Programming.

The first four phases of the program include the basics of computer systems, the inner details of program creation and execution, essentials of high-level programming languages, including variables, data types, conditionals, loops, functions, arrays, simple aggregation mechanisms. Introduction to classes, inheritance, virtual functions, and polymorphism.

Covering the essentials of system design and a smooth introduction to design patterns with several practical examples. A deep introduction to algorithms and data structures, including the standard library implementations (both in C++ and Java).
